Get rid of boring submit messages animations on Contact Form 7. Have nice animations upon submission. 

== Description ==

Now you can customise particle effects animations on the submit event of contact form 7. Choose from the wide range of available animation options.

== Installation ==

1. Upload the entire `cp7-submit-animations` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
You will find ***CP7 Submit Animations*** menu in your WordPress admin screen.

Within ***CP7 Submit Animations*** you will find several fields like Type, Stype, Direction, Color etc., which can be tweaked to get the desired submit animation for Contact Form 7.

== Frequently Asked Questions ==
 
= Would the Plugin work even if I don't provide any values in the plugin settings? =
 
Yes the plugin works right off the bat with default values.
 
= What are the default values? =
 
Type = circle
Color = black
Duration = 0.5s
Size = 3
Style = fill
Direction = left
